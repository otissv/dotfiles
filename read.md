#Dotfiles for Arch linux and i3wm

##Prerequisite

Required
- Arch linux
- i3 window manager
- i3Blocks
- dmenu-xft-mouse-height
- dmenu_extended
- ttf-font-awesome
- ttf-font-iconss

Optional
- ttf-droid
- ttf-droid-monovar
- Music player daemon (mpd)
- zsh
- blueman
- wifi menu
- alsamixer
